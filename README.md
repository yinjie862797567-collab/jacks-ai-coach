# 杰克船长AI智能体

覆盖19个细分行业、60+创业工具，每个场景都有专业AI教练陪你实操。

## 使用方式

浏览器打开即可使用，选择细分行业和工具开始对话。

## 环境变量

| 变量 | 说明 |
|------|------|
| `MIMO_API_KEY` | MiMo API密钥（推荐） |
| `DEEPSEEK_API_KEY` | DeepSeek API密钥（备选） |

至少设置一个API Key才能使用对话功能。
